# 1.  <a name='first'></a> 上海初雪
<br><br>&emsp;&emsp;今天看到温度很低,出门时还在纠结是骑车还是乘地铁.提着垃圾袋站在路口,突然有个凉凉点落在脸上,不自觉就漾出笑来.还看了眼手机,明明昨天报导有雪但是没有下,今天却有些征兆.丢完垃圾我开锁跨上车走向那条走了三年的路,迎接着这三年的第一场雪.
<br><br>&emsp;&emsp;刚开始大概是:"撒盐空中差可拟",他们都很细小,只是凭着风大来刮痧着我的脸. 也是一阵一阵的出现,当然 这些也是风的原因.我想他们落在外套上也许会留下踪迹,可气温还是不太允许他们这个年岁的雪花崽崽被人捕捉. 等到快到公司了,他们也越来越紧密,地上也显出湿漉漉的征兆,这时候看着深色的高架路的底部或者远眺会发现他们活蹦乱跳的在空气中飞舞,乱窜.锁好车走上楼的时候还在想:也许今天就这样了.
<br><br>&emsp;&emsp;谁知道呢,在我接完一杯水的功夫他们就已经完成了集结--开始成片的出现在窗外.这时候却像"未若柳絮因风起"了."这才叫雪么"我一边这样想着一边再关小一点窗缝.这真的不能让人真心工作,看着窗外的雪,好想这才是我来这里的第一个冬天,我在这里进行的第一个循环才将要宣告结束.慢慢同事都来的差不多了,都在说着积不了雪,都在边做边往窗外看. 今天风真大啊,风真的好大,从云中扯出这么多白絮,多像老家春天的杨绵和柳絮啊. 好小啊....这已经是他最大的力气了.


# 2.  <a name='second'></a> 吉林的雪
<br><br>&emsp;&emsp;想起来吉林的雪,我最爱的雪.他们满足你对雪的所有幻想,似乎之前所有对雪的描写都是最朴素的白描--他们就是那样.像鹅毛,能一瞬间开满树白花,能吃下整个鞋子,一个操场能推出好多个一人高的雪球,能埋进去一整个人,能用雪堆出等比例的小汽车,随手一团就是洁白的雪球.他不仅是白色的,他是早退日光的最后一丝橘黄,他是夜幕前奏的深蓝,他也是没来得及冬眠的花坛中的青绿,他比春天的花更加动人,当然也冻人. 
<br><br>&emsp;&emsp;我想起来每一条没有脚印的雪路,我也想起来扫雪,松鼠和柏油路.我想起来每天都有个大爷在雪中倒立,我想起来十月份接到的第一份雪花,想起来从教室出来看到被蓝色雪景映着的风华碑,想起来那颗心形的大树从绿色变成白色,想起来超市的灯照出路上的一颗颗大小不一的脚印.甚至那次去雪场大家一起练习滑雪的笑脸.原来体育老师真的会一边激动的在远方吼着什么,最后看我倒下瞬间的扶额皱眉.哈哈哈 好开心吉林的雪 说不出写不出 好多的东西在脑海里, 他是幸福的快乐的激昂的.

# 3.  <a name='third'></a> 家的雪
*这部分几乎就跟雪无关了*
# 3.1. <a name='thirdPOne'></a> 开封
<br><br>&emsp;&emsp;那会还是在姑姑家，姑父养了条德牧叫哈利。虽然我小但我一开始就不怎么怕他的，因为跟我爸那些细狗比哈利他看起来可人多了。为什么先说哈利呢？因为那年冬天姑父用烧锅的木材给我们做了个小雪橇板，然后给哈利套上让哈利拉雪橇。雪堆满了胡同，我们被姑父叫到门口，看着他做的小雪橇板我们仨都哇哇的老激动了。他怎么给哈利套的绳子我记不清了，我只记得哈利他这家伙老往墙沿跑，我们一个一个上去试，然后我们一个一个都翻倒在斜对门邻居的门口。”这哈利！咋光歪着跑！“，姑父往哈利那赶前两步扬起手，哈利一点躲的意思都没有，伸着舌头大口大口的吐着白雾。我们小孩倒是很开心。
<br><br>&emsp;&emsp;我不记得那时候打雪仗的伙伴都有谁和细节了，不记得有没有堆过雪人，关于雪的只记得小雪橇了，幸好还记得小雪橇。

# 3.2. <a name='thirdPTwo'></a> 孙陆
<br><br>&emsp;&emsp;我想起来雪人，好几个雪人，说个最大的吧。
<br><br>&emsp;&emsp;那是我八九岁大雪快化的时候。过完年了餐馆要收拾收拾开门。然而那年雪积的巨多，加上温度有些回升，房顶的雪也全部掉到院子里。因为还有厨房和房间需要打扫所以我妈就只是把雪堆到院子中间搞了个两米高的雪坡，加上我和弟弟团的大雪球它就成了雪巨人。
<br><br>&emsp;&emsp;院子几乎还是被雪巨人满占着但已经留出环形的一条路也没功夫管他了。大家开始打扫房间，烧水洗刷工具桌椅窗户。厨房那时候还是砖地，平常的油污滴在地上很容易形成钟乳球的东西让整个地板像是一个凝固的黑泥潭。我和我爸蹲在地上戗子，我妈和我舅收拾厨房的杂物，还粉刷了大厅，好像还砸了什么东西，砌了什么东西，记不太清了，反正所有人都忙了一整天。下午的时候谁抬过来一箱蒙牛纯牛奶，重来受不了纯奶的我一口气喝了一袋，好清凉，好解渴。两天吧应该，忙活完了想起来院子里的雪巨人，可是雪已经有些硬化成冰了根本处理不动，两三个星期才化的差不多。
