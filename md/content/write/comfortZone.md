## 1. <a name='first'></a>生活习惯
*所谓缺乏激情*
*慵懒*

```
需要破除的
不再每天洗当天的袜子
不再观察现实中的色彩结构线条
不再学习
不再坚持

需要保持的
早起
鸡血
节食
```

```
不读书
旧的价值观

需要保持的
质疑
灵光一闪
```
